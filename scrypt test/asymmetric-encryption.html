Asymmetric Encryption (Public-Key Cryptography)

Asymmetric encryption uses a pair of mathematically linked keys: a public key (which can be freely shared) and a private key (which must be kept secret). Data encrypted with the public key can only be decrypted with the corresponding private key. This tool demonstrates RSA encryption/decryption and Diffie-Hellman key exchange.
