Hash Functions

Hash functions are one-way cryptographic algorithms that transform data of any size into a fixed-size string of characters, called a hash value or digest. They are crucial for ensuring data integrity, digital signatures, and password storage, as even a small change in the input data results in a drastically different hash value.
